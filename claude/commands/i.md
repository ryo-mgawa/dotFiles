# プロジェクト初期化 (日本語版)

プロジェクトを日本語のCLAUDE.mdガイドで初期化します。

既存のCLAUDE.mdファイルがある場合は上書きして、以下の内容でCLAUDE.mdファイルを作成してください：

```markdown
# プロジェクトガイド

## 概要
このプロジェクトについての基本的な情報を記載してください。

## 開発環境
- 必要なツール、言語、フレームワーク
- インストール手順
- 環境設定

## よく使用するコマンド
- `npm run dev` - 開発サーバーを起動
- `npm run build` - プロダクションビルド
- `npm run test` - テストを実行
- `npm run lint` - コードの品質チェック

## コードスタイル
- コーディング規約
- 命名規則
- フォーマット設定

## アーキテクチャ
- プロジェクトの構造
- 重要なパターンや設計方針
- ディレクトリ構成

## 開発フロー
- ブランチ戦略
- コミット規則
- レビュープロセス

## 注意事項
- 重要な制約事項
- 避けるべき実装パターン
- セキュリティに関する注意点

## 参考リンク
- ドキュメント
- 外部リソース
```

プロジェクトの特性に応じて、適切な内容に調整してください。
